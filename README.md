# Cork Legal Services - Demo Project

A high-fidelity landing page demonstration for a legal firm based in **Cork & Mallow**. This project showcases modern UI/UX principles, responsive design, and professional service presentation.

## 🔗 Live Demo
You can view the live site here: 
**[Visit Live Site](https://kovtonukanatol-cloud.github.io/cork-legal-demo/)**

## 🛠️ Features
- **Responsive Layout:** Optimized for desktop, tablet, and mobile devices.
- **Modern UI:** Clean, trustworthy aesthetic using Navy and Gold accents.
- **Dynamic Components:** Includes interactive service cards, testimonials, and a functional-looking contact form.
- **Performance:** Lightweight code with optimized Google Fonts integration (Inter & Playfair Display).

## 📁 Project Structure
* `index.html` — The main structure and content of the site.
* `README.md` — Project documentation and setup instructions.
